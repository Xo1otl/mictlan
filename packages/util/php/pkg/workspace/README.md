# workspace

workspaceを抽象化して考え、モジュール化したもの
いつかパッケージ化したい

## 仕様
- `location`
    - workspaceという概念の中で仕様される抽象的な位置
    - ファイルパスだけでなく、外部リソースへのリンクなども扱うことが可能
- `root`
    - 木構造になっているlocationにおけるルート
