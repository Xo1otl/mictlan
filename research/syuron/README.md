## SHGの変換効率のシミュレーションによる計算

- 層ごとに電解分布が求まる
- 一定の波長&モードで各層を光が伝搬し、層が異なるためそれぞれで異なるSHGが発生する
- SH光どうしが重なりあって出力になる、それ計算したら変換効率がわかる
- 電解分布を求める部分から波長が絡んできており、波長依存性を理論的にとくのは難しく、シミュレーションで出すと楽
