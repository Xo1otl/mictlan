# 順方向の計算は転送行列法ですぐ終わる

1. 一個消去するとNCMEが一個に二階の微分方程式になる、それを解くとL^2の関数がでてくる、それが一つのドメイン
2. 転送行列で計算すると、異なる幅のドメインを通して、電界の位相と振幅が変化
