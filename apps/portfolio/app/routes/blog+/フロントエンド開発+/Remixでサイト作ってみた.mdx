## Katexについて

Katexは数式を綺麗に表示するためのライブラリです。

### Example

```math
\frac{d}{dz} A^{\omega}(z) = -j\kappa^{*}[A^{\omega}(z)]^{*} A^{2\omega}(z) \exp(-j(2\Delta)z)
```

```math
\frac{d}{dz} A^{2\omega}(z) = -j\kappa[A^{\omega}(z)]^{2} \exp(+j(2\Delta)z)
```

```math
2\Delta = \beta^{2\omega} - 2\beta^{\omega}
```

```math
\kappa = \frac{2\omega \epsilon_0}{4} \iint [E^{2\omega}(x,y)]^{*} d(x,y)[E^{\omega}(x,y)]^{2} dxdy
```
