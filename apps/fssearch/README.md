# fssearch

- 全文検索エンジンをバックエンドとして使用し、ファイルシステムに対する全文検索ができるようにする
- meilisearchを全文検索エンジンとして使用
- とりあえずUI無しでサーバーとして実装する
- UIをvscode拡張にする場合はscriptsに書くかもしれない

## 考慮する点
- indexが必要
    - 変更に応じてindexの場合
        - ずっとツールを起動し続け、監視を続ける必要性がある
    - 横着する場合
        - gitを使用して差分を取る等考えられるがかなり複雑化しそう
